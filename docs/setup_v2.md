# Updated Local Setup Guide

## Requirements

- Node.js v18+
- npm v9+

## Setup Steps

1. Clone the repository
2. Install dependencies:
   npm install
3. Create a .env file:
   API_URL=https://api.newservice.com
4. Start the app:
   npm run start

## Notes

- Node upgraded from v14 to v18
- npm run dev replaced with npm run start
