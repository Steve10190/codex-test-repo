# codex-test-repo

## Local Setup

1. Clone the repository
2. Install dependencies:
   npm install
3. Create a `.env` file:
   API_URL=https://api.newservice.com
4. Start the app:
   npm run start

## Requirements

- Node.js v18+
- npm v9+

## Notes

- Node.js was upgraded from v14 to v18+
- The startup command is `npm run start`
